# apriori-proto

Protocol definitions and Data Transfer Objects (DTOs) for APriori authentication and communication.

## Overview

This crate provides common data structures and protocol definitions used across the APriori ecosystem, including:

- Authentication DTOs (challenge, auth, refresh requests/responses)
- Role definitions
- Other protocol-level types

## Usage

Add this to your `Cargo.toml`:

```toml
[dependencies]
apriori-proto = { git = "https://github.com/apriori-network/apriori-proto.git", branch = "main" }
```
